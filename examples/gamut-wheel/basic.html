About This Example

  • This example demonstrates the Gamut Wheel with Philips Hue Gamut C (latest generation bulbs).
  • The wheel displays only colors that can be physically reproduced by Gamut C bulbs.
  • Notice how highly saturated colors are automatically mapped to displayable colors using perceptual gamut mapping.
  • The XY coordinates shown are ready to use with the Philips Hue API.
  • Brightness is shown as a percentage (0-100) and should be converted to 0-254 for the Hue API.
