Understanding Gamut Differences

  • This example compares how the same color selection is mapped across different Philips Hue gamuts.
  • Try selecting a highly saturated color (like bright magenta or cyan) and observe how each gamut handles it differently.
  • Gamut A (oldest) has the most restrictive color space and will reduce saturation the most.
  • Gamut C (latest) offers the widest range and preserves more saturated colors.
  • Notice how the XY coordinates differ for each gamut, even when selecting the same HSV color.
  • The actual displayed color (hex value) will vary based on how each gamut maps the color to its available range.
